Payment & Resend Configuration

The checkout verifies Paystack payments on the backend and sends notifications through Resend. Configure secrets in environment variables before going live.

Paystack Secret

Set PAYSTACK_SECRET_KEY on the server. Payments stay pending until backend verification succeeds.

Admin Login

Set CREATORLIFT_ADMIN_PASSWORD or ADMIN_PASSWORD. Hard-coded admin passwords are no longer used.

Resend Email

Set RESEND_API_KEY, RESEND_FROM_EMAIL, ADMIN_EMAIL, and NEXT_PUBLIC_APP_URL. If Resend fails, campaign actions still complete and the error is logged.

Supabase Storage

For Vercel, set SUPABASE_URL and SUPABASE_SERVICE_ROLE_KEY. Without them, CreatorLift falls back to db.json, which is best for local or Render testing.
